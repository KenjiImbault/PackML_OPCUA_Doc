Building and starting the server

  • In Visual Studio, stop the server if it's launched.
  • On the top, you should see these buttons: Debug - Any CPU - Start
  • Click on Debug and select Release instead
  • Click on Build>Build Solution.
  • In the project folder, go to bin>Debug, and copy the Data folder
  • In the project folder, go to bin>Release, and paste the Data folder
  • The executable is FIP.PackMLModel.exe, double click on it to launch the server.
  • The server is now launched, and can be accessed with any client, such as UaExpert
