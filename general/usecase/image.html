View the state machine

  • In UaExpert with the client connected to the server, click on Document>Add.
  • Select Image Viewer and click Add
  • Now develop the machine where you want to see its state machine
  • Locate its MachineImage variable, and drag and drop it into the image viewer
  • The image will update in real time. Meaning that if the machine changes state, the highlightned state will change as well.
