Stack light csv file

In the "Data" folder, you can find an example on how to format the .csv files for the stack lights called "_Template_StackLights.csv"
In more detail (Please note that the first line is reserved for headers):

  • The first column contains the stack light ID of each stack light.
    Each ID is unique. (However it must only be unique in the same file. Meaning you can have one ID "1" in file A and one ID "1" in file B)
  • The second column contains the description of the stack light
Reading this file using a method in an instance will add all of the stack lights in the instance.
