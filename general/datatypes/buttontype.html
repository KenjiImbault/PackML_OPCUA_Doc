Data Type: ButtonType

This data type describes a button.
The button is defined with 3 components:
  • The ID (int) of the button. (The ID is unique for each machine, however two machines can have different buttons with the same ID.)
  • The name (string) of the button. (For example: "Pause/Resume" or "Start".)
  • The commands (int[]) of the button. It's a list of int with each int representing a transition number. The transition number corresponds to the PackML Tag Values. (Check the cheatsheet.)
    If a button is triggered, it will try to apply the first transition. If it can't apply the first transition, it will try the second one. So on and so on until a transition is applied, or if no transition can be applied: nothing happens.
