Data Type: AlarmType

This data type describes an alarm.
The alarm is defined with 4 components:
  • The ID (int) of the alarm. (The ID is unique for each machine, however two machines can have different alarms with the same ID.)
  • The message (string) of the alarm to be displayed when the alarm is triggered. (For example: "Error - No more paper")
  • The transition (int) to be applied to the machine if the alarm is triggered. The transition number corresponds to the PackML Tag Values. (Check the cheatsheet.)
  • The state (true/false) of the alarm. If the state is "true", then the alarm is triggered. Otherwise it is off.
