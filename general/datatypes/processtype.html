Data Type: ProcessType

This data type describes a process.
A process is a series of transition to apply after a certain ammount of time.
A process is defined with 3 components:
  • The number of loops (int). This corresponds to the number of time to execute the transitions.
  • The commands (int[]) to apply. Each transition will be applied in sequential order. (For example {2, 10, 1} will execute Start -> Complete -> Reset to the machine)
  • The time to execute a StateCompleted transition (int).
  • The time to execute any other transition (int).
