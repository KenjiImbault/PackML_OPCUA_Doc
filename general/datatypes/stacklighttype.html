Data Type: StackLightType

This data type describes a stack light.
The stack light is defined with 3 components:
  • The ID (int) of the stack light. (The ID is unique for each machine, however two machines can have different stack lights with the same ID.)
  • The description (string) of the stack light. (for example: "Solid blue light" or "Beeping horn".)
  • The state (true/false) of the stack light. If the state is "true", then the stack light is triggered. Otherwise it is off.
