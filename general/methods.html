string delimiter: the delimiter of the .csv file. Usually a comma.

AddStackLightsFromCSV(string filename, string delimiter) Read a well formated .csv file and add the stack lights to the machine.
See "_Template_StackLights.csv" in the Data folder to see how to correctly format a .csv file.		 string filename: the name of the file.
string delimiter: the delimiter of the .csv file. Usually a comma.
ExecuteProcess(ProcessType process) The machine will execute the first command of the process,
it will then wait and then try to execute a StateCompleted transition,
before trying to execute the second command after waiting a certain amount of time, so on and so on.		 ProcessType process: The process to apply to the machine.
int[] GetAvailableCommands() Output all the available commands to apply to the machine int[] commands: The available commands
int GetNext(int command) Outputs the state of the machine if the command given as an input were applied to the machine.
Throws an exception if the command is not applicable.		 int command: The command to be applied. int state: The state of the machine if the command were to be applied.
ReadDataFromDB(string DBFileName) Load the data from the database file into the machine. string DBFileName: the name of the file.
RemoveAlarm(int alarmID) Remove the alarm corresponding with the alarm ID from the machine. int alarmID: The ID of the alarm to remove.
bool IsCommandAvailable(int command) Returns True if the given command is available. False otherwise. int command: The command to test. bool val: True if the command is available, False otherwise.
int MoveNext(int command Applies the command to the machine and changes its state.
If the command is not applicable, throws an error.		 int command: The command to apply. bool state: The new state of the machine.
ReadDataFromCSV(string filename, string delimiter) Load the data from a .csv file to the machine.
See "__Template_Machine.csv" in the Data folder to see how to format the file correctly.		 string filename: the name of the file.
string delimiter: the delimiter of the .csv file. Usually a comma.
RemoveButton(int buttonID) Remove the button corresponding with the button ID from the machine. int buttonID: The ID of the button to remove.
RemoveStackLight(int stacklightID) Remove the stack light corresponding with the stack light ID from the machine. int stacklightID: The ID of the stack light to remove.
SafeMoveNext(int command) Applies the command to the machine and changes its state.
If the command is not applicable, do nothing.
The returned status code will change depending on whether the transition could be applied or not		 int command: The command to apply.
SaveDataToCSV(string csvFileName, string delimiter) Save the data of the machine to a .csv file that can then be loaded back. string csvFileName: the name of the file.
string delimiter: the delimiter of the .csv file. Usually a comma.
SaveDataToDB(string DBFileName) Save the data of the machine to a .db file that can then be loaded back. string DBFileName: the name of the file.
SetMachineName(string name) Change the name of the machine to the given input. string name: The name of the machine.
SetState(int state) Set the state of the machine to the given input int state: The new state of the machine.
TriggerAlarm(int alarmID Triggers the alarm with the corresponding id. int alarmID: The ID of the alarm to trigger.
TriggerButton(int buttonID) Triggers the button with the corresponding id. int buttonID: The ID of the button to trigger.
TriggerStackLight(int stacklightID) Triggers the stack light with the corresponding id. int stacklightID: The ID of the stack light to trigger.
UntriggerAlarm(int alarmID) Untriggers the alarm with the corresponding id. int alarmID: The ID of the alarm to turn off.
UntriggerStackLight(int stacklightID) Untriggers the stack light with the corresponding id. int stacklightID: The ID of the stack light to turn off.
